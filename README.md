# Scripts Shell para Ubuntu

Este repositório contém uma coleção de scripts em Bash para manutenção e otimização do sistema Ubuntu.  
Inclui ferramentas para limpeza de cache, esvaziamento de lixeira, liberação de memória RAM e remoção de pacotes órfãos.

## Scripts incluídos

- `limpeza-ubuntu.sh` — Limpeza completa do sistema e do usuário
- `memoriaram.sh` — Liberação de memória RAM

## Requisitos

- Ubuntu 20.04 ou superior
- Bash
- Permissões de sudo

## Uso

```bash
sudo bash limpeza-ubuntu.sh --wipe-history
bash memoriaram.sh
